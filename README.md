# The Big Tech Personality Audit

A snarky, illustrated field guide to 27 tech companies — archetypes, burns, glow-up suggestions, and three quadrant matrices.

## View it live

After enabling GitHub Pages, this site is at: `https://YOUR_USERNAME.github.io/tech-personas/`

## To run locally

Just open `index.html` in a browser. No build step, no dependencies.

## Adding the photos

Drop your generated animal photos into the `/images/` folder using the filenames listed in `images/README.md`.

## Structure

- `index.html` — the whole site, all 28 paginated pages
- `images/` — animal photos (one per company)
